# Baseball Stats

##Overview
This project is a collection of Python classes and corresponding PostgreSQL components developed in order to scrape data off baseball statistic websites and load into an OLAP-ish database structure

##Pre-Requisites
* Python 2.7.10
* PostgreSQL
* psycopg2
